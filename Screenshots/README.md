# Dashboard Screenshots

This folder contains screenshots of all 3 dashboard pages.

## Required Screenshots:

1. **page1_sales_overview.png** - Sales Overview page with KPIs, charts, and map
2. **page2_product_analysis.png** - Product Analysis page with product performance
3. **page3_customer_insights.png** - Customer Insights page with customer data

## How to add screenshots:

### For each page:

1. Open Power BI Desktop
2. Open `SalesAnalytics.pbix`
3. Go to the specific page (Sales Overview, Product Analysis, or Customer Insights)
4. Press `Win + Shift + S` (Windows Screenshot)
5. Select entire dashboard page
6. Paste in Paint or image editor
7. Save with appropriate name:
   - `page1_sales_overview.png`
   - `page2_product_analysis.png`
   - `page3_customer_insights.png`
8. Place all 3 files in this folder (`Screenshots/`)

## Recommended dimensions:
- Width: 1920px
- Height: 1080px
- Format: PNG
- Quality: High (for presentation)

## Alternatives:
You can also use:
- Snagit
- Greenshot
- Windows Snipping Tool
- Power BI's built-in Export (File → Export → Export to image)

## After adding screenshots:

Upload to GitHub:
```bash
git add Screenshots/
git commit -m "Add all 3 dashboard page screenshots"
git push
```

**Note**: All 3 screenshots will be displayed in README.md file.
